# 🚀 Personal Portfolio Website

Welcome to the source code of my personal portfolio website! This is a modern, highly interactive, and fully responsive web application built to showcase my skills, projects, and professional journey.

## 🔗 Live Demo
**[Click here to view the live portfolio!]https://fahmidhaabdulhakeem-dev.github.io/PORTFOLIO/

## ✨ Features
- **Premium UI/UX:** Designed with a sleek "Glassmorphism" aesthetic.
- **Interactive Animations:** Features a dynamic typing effect, custom animated cursor, and scroll-reveal transitions.
- **3D Tilt Cards:** Project and skill cards that react to mouse movements using vanilla JavaScript.
- **Fully Responsive:** Looks great and functions flawlessly on desktops, tablets, and mobile devices.
- **Modern Styling:** Utilizes custom CSS variables, animated mesh gradients, and premium typography.

## 🛠️ Technologies Used
This project was built from scratch without the use of heavy frameworks, ensuring blazing-fast performance:
- **HTML5** (Semantic structure)
- **CSS3** (Variables, Flexbox, CSS Grid, Keyframe Animations)
- **Vanilla JavaScript** (DOM manipulation, Intersection Observer API, Event Listeners)
